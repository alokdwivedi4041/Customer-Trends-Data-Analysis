# 📊 Customer-Trends-Data-Analysis

## 🔍 Overview

This project demonstrates a complete end-to-end data analytics workflow, starting from data loading to delivering business insights through dashboards and reports. It covers data preprocessing, exploratory data analysis (EDA), SQL querying, and visualization using industry-standard tools.

The goal of this project is to extract meaningful insights from raw data and present them in a clear, actionable format for decision-making.

---

## 📁 Dataset

* The dataset used in this project contains structured data relevant to business analysis.
* It includes multiple features such as categorical and numerical variables.
* Data was initially loaded using Python for preprocessing and analysis.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
* **PostgreSQL** – Writing and executing SQL queries
* **Power BI** – Dashboard creation and visualization
* **Gamma** – Presentation (PPT) creation
* **Jupyter Notebook** – Development environment

---

## ⚙️ Steps Involved

### 1. Data Loading

* Imported dataset using Pandas
* Inspected structure, columns, and data types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Standardized column formats

### 3. Exploratory Data Analysis (EDA)

* Performed univariate and bivariate analysis
* Identified trends, patterns, and outliers
* Visualized key metrics using charts and graphs

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Wrote SQL queries to:

  * Aggregate data
  * Filter and sort records
  * Perform joins and ranking
* Extracted insights for reporting

### 5. Dashboard Creation (Power BI)

* Built an interactive dashboard
* Included KPIs, charts, and filters
* Focused on user-friendly and insightful design

### 6. Reporting & Presentation

* Created a structured report summarizing findings
* Developed a professional presentation using Gamma
* Highlighted key insights and recommendations

## ▶️ How to Run

1. **Clone the repository**

   ```bash
   git clone <your-repo-link>
   ```

2. **Install required libraries**

   ```bash
   pip install pandas numpy matplotlib seaborn psycopg2
   ```

3. **Run Jupyter Notebook**

   * Open the notebook file
   * Execute cells step-by-step for EDA and preprocessing

4. **Set up PostgreSQL**

   * Create a database
   * Import cleaned dataset
   * Run SQL queries

5. **Open Power BI**

   * Load dataset or connect to PostgreSQL
   * Open the dashboard file (.pbix)

6. **View Presentation**

   * Open the Gamma-generated PPT for final insights
